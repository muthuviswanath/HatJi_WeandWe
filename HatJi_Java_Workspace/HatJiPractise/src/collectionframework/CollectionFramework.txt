List:
	1. Can store duplicates
	2. It is an index based collection
	3. Random access is possible
	4. Can store more than one null values

ArrayList:
	1. It maintains the insertion order
	2. It is a non-synchronized class
	3. It internally uses (dynamic/growable) array.
	4. Data is stored in the form of Array.
	5. Initial capacity and incremental capacity are applicable
	6. Default capacity is 10 elements
	7. When the 11th element is stored the capacity inrements as follows
			((Current Capacity) * 3)2 + 1
				(10 * 3)/2 + 1
				= 16
	8. It is not possible to control the growth.
	9. It involves shift operations while inserting and deleting the values
	10. Consumes less memory
	11. Memeory is continuous
	12. Three Overloaded constructors:
		ArrayList(int initialCapacity)
		ArrayList()
		ArrayList(Collection<? extends E> c)
	
	Syntax:
		ArrayList <Generic> arraylist_name = new ArrayList <Generic> ();
		ArrayList <Generic> arraylist_name = new ArrayList <Generic> (initial_capacity);
		ArrayList <Generic> arraylist_name = new ArrayList <Generic> (collection_name);
